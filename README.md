# slack-irc-echobot

IRC에서 Slack으로, Slack에서 IRC로 그대로 전달해주는 봇

```
# 현재 채널을 echo할 채널로 설정
?setechohere
# 다음 닉네임들이 하는 말은 무시
?setignore <nick1> <nick2> ...
# 다음 닉네임을 더이상 무시하지 않음
?unsetignore <nick1> <nick2> ...
```

### Requirements

- slackclient
